# Angband-279

This repository now contains:

- `angband-2.7.9v6.tar.gz` — original Angband 2.7.9v6 source archive.
- `angband-2.7.9v6/` — extracted original C source tree.
- A native Swift terminal reimplementation prototype in `Sources/main.swift`.

## Run the Swift version

```bash
swift run angband279-swift
```

## Controls

- `w`, `a`, `s`, `d`: move
- `.`: wait
- `q`: quit

Descend stairs (`>`) to reach deeper levels. Survive to depth 8 to win.
